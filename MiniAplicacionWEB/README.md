#Proyecto Web

Desarrollo de Aplicaciones Web  
Alumno: Angel Daniel Mendoza Collazo

El proyecto utiliza una aquitectura cliente-servidor.  
Cliente: Interfaz desarrollada con HTML y validaciones en JavaScript (`/views`).  
Servidor: Lógica de negocio procesada en PHP (`/php`) y persistencia en MySQL (`/config`).

EL objetivo es utilizar la comunicación entre cliente y servidor, para poder registrar usuarios en una base de datos, así como ver y manipular los usuarios que estén registrados.

Este proyecto pordría resolver la problemática de un registro de usuarios en una empresa o en algún pequeño negocio, pudiendo registrar por ejemplo algún proveedor y/o comprador.

Para este proyecto se utilizaron tecnologías variadas como Visual Studio Code, así como el lenguaje PHP, usamos PHP MyAdmin Como base de datos para almacenar todo esto, usamos XAMPP para poder utilizar las tecnologías de Apache y PHP MyAdmin.

Aplicamos algunos conceptos como el de la seguridad, en este caso la seguridad es proporcionada por SHA1, para autenticar los datos de los usuarios que se registren.

Hubieron varias enseñanzas de este proyecto, como conectar las bases de datos, ligar los códigos para que funcionen, registrar los usuarios, y verlos en la base de datos, podría mejorar la comprensión sobre todos los conceptos que utiliza por ejemplo SQL para su codificación, y hacerlo un poco más rápido.
